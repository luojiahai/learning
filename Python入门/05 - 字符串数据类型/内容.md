## 字符串

字符串（String）是一种数据类型，由多个字符（Character）组成，用英文的双引号`"`或单引号`'`包括起来。Python支持中文字符，所以我们在上一章可以打印出`你好，世界！`这个字符串。例子如下：

```python
>>> sentence = '你好，哔哩哔哩！'
>>> print(sentence)
你好，哔哩哔哩！
```

特定的操作符运用在字符串上会有特定的功能，譬如`+`、`*`。`+`的功能是字符串串联（String Concatenation），`*`的功能是字符串复制（String Replication），例子如下：

```python
>>> 'ha' + 'ha'
'haha'
```
```python
>>> 'ha' * 3
'hahaha'
```
